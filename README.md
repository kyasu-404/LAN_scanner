# LAN_scanner
Возможности: ARP-обнаружение, 3 режима сканирования, многопоточность, определение ОС, определение версии сервисов, умная классификация устройств, определение производителя по MAC, сортировка по IP, экспорт результатов (можно выбрать терминал, CSV, JSON, HTML), Deep Scan одного IP, возможность указать свои порты

УСТАНОВКА
1. Установить NMAP:
https://nmap.org/download.html

2. Добавить его в PATH

3. Установить Python-библиотеки:
py -m pip install python-nmap rich

4. Запустить скрипт:
py LAN_scanner.py
